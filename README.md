# Little Bit of This, Little Bit of That Extension

## Description

This project is going to have two main interferences in a webpage. This extension will bring the
brainrot that you so love to you.

## Features

### Flashbangs

Meme flashbangs every set interval. Must click the page upon load to activate audio.

### Wall of 67

A pop up where you need to click the correct 67 to close the popup.

### Mr Peebles

The glory of Mr. Peebles is upon you. If you want to close it, you have to flip a coin. When you
land heads, then the popup will close.
